# Danish Construction Industry Analysis (2015–2024)

This repository documents my **personal learning project** in MySQL, Excel, and Power BI, using Denmark’s construction industry as a real-world scenario.  
The goal is to practice querying, visualization, and data storytelling while showcasing progress in applying these tools.

---

## Purpose
- Practice SQL queries for aggregating and analyzing turnover data  
- Build Excel charts to visualize trends and sector comparisons  
- Explore Power BI dashboards for interactive analysis  
- Showcase my learning journey in data workflows and visualization  

⚠️ **Note:** The results and visualizations presented here are **not professionally developed documents** and should not be used for citation or official purposes.  
This is strictly a **personal project for learning and practice**.

---

## Analyses Covered
- **Total Turnover by Year (2015–2024):** Line chart showing industry growth and peak years  
- **Industry Ranking by Turnover (2024):** Bar chart comparing revenue across construction sectors  
- **Share of Industry Segments (2024):** Percentage-based ranking of sector contributions  
- **Housing vs Other Trends in New Buildings and Extensions (2015–2024):** Dual-line chart showing investment shifts  
- **Repair vs New Construction Mix (2015–2024):** Multi-line chart comparing turnover across categories  
- **Work Type Composition (2024):** Pie chart breaking down percentage shares of work types  

---

## Tools Used
- **MySQL Workbench** → Querying and aggregating turnover data  
- **Excel** → Creating static charts and formatting  
- **Power BI** → Building interactive dashboards
